These are the instructions on how to run the game:
1. Download and install Visual Studio Code by following this link: https://code.visualstudio.com/
2. Download and install the Java Development Kit (JDK) via this link: https://www.oracle.com/eg/java/technologies/downloads/
3. Setup Java on Visual Studio Code by following the instructions found through this link: https://code.visualstudio.com/docs/languages/java
4. Open Visual Studio Code.
5. Go to "Terminal" -> "New Terminal".
6. Compile the Java file (.java) by typing "javac Game.java" and pressing "Enter".
7. Run the Java byte code (.class) by typing "java Game" and pressing "Enter".
8. The game will start, and you will be prompted to enter a move.

According to my knowledge, there are no implementation issues. However, there are certain arrangements of the board that make the game impossible to solve.
